# Civilization 6 Settings Generator

Simple Shell Script that generates a Settings Configuration for a Civilization 6 Game based on some defined chances.

## Generated Settings

- Game Mode
- Map Type (WIP)
- Map Size (WIP)
- CIV (WIP)

## Chances

The defined Chances can be found in the Script.
